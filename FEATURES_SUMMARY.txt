QUICK START COMMANDS
================================================================================

1. Copy Files:
   cp frontend/src/services/*.ts your-app/frontend/src/services/
   cp backend/api/error_reporting_api.py your-app/backend/api/

2. Register API:
   Edit backend/server.py:
   from backend.api.error_reporting_api import router
   app.include_router(router, prefix="/api/admin")

3. Initialize Services:
   await WiFiConnectionService.getInstance().initialize()
   await pinAuthService.initialize()
   ErrorReporter.getInstance().initialize()

4. Run Tests:
   SelfTestService.getInstance().runAllTests()
